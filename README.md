# 🕌 Jadwal Shalat Modern (PWA)

Aplikasi jadwal shalat berbasis web yang modern, ringan, dan responsif. Dibangun dengan fokus pada kecepatan performa dan estetika **Material Design 3**.

[![Live Demo](https://img.shields.io/badge/demo-GitHub%20Pages-006d44?style=for-the-badge&logo=github)](https://dayxnight.github.io/jadwal-shalat/)
[![Alpine.js](https://img.shields.io/badge/Alpine.js-3.x-blue?style=for-the-badge&logo=alpine.js)](https://alpinejs.dev/)
[![PWA](https://img.shields.io/badge/PWA-Ready-orange?style=for-the-badge&logo=pwa)](https://web.dev/pwa/)

---

## ✨ Fitur Utama
- [x] **Material Design 3 UI** – Antarmuka bersih dengan palet warna harmonis.
- [x] **PWA Ready** – Bisa diinstal di Android/iOS dan mendukung mode offline.
- [x] **Deteksi Lokasi Otomatis** – Menggunakan Geolocation API untuk akurasi jadwal.
- [x] **Animasi Halus** – Transisi elegan berbasis CSS murni (Bebas GSAP/Library berat).
- [x] **Reaktif dengan Alpine.js** – Logika UI yang ringan dan instan.
- [x] **Tema Gelap & Terang** – Menyesuaikan kenyamanan mata pengguna.
- [x] **Countdown Real-time** – Menampilkan sisa waktu ke jadwal shalat berikutnya.
- [x] **Kalender Hijriah** – Informasi tanggal Islam yang sinkron otomatis.
- [x] **Efek Luminous** – Highlight bercahaya pada jadwal yang sedang aktif.

## 🛠️ Teknologi yang Digunakan
- **Logic:** [Alpine.js 3.x](https://alpinejs.dev/)
- **Styling:** Vanilla CSS dengan variabel M3
- **Animations:** Pure CSS Keyframes
- **API:** [equran.id](https://equran.id/) & [Aladhan API](https://aladhan.com/prayer-times-api)
- **Deployment:** GitHub Pages

## 🚀 Cara Menjalankan Secara Lokal
1. Clone repositori ini:
   ```bash
   git clone https://github.com/dayxnight/jadwal-shalat.git
   ```
2. Buka folder proyek:
   ```bash
   cd jadwal-shalat
   ```
3. Jalankan menggunakan server statis (misal: Live Server di VS Code).

## 📌 Catatan Pengembangan
Proyek ini telah melalui tahap optimasi besar-besaran:
- Menghapus ketergantungan pada library animasi eksternal untuk performa maksimal.
- Menggunakan sistem variabel CSS ringkas untuk kemudahan kustomisasi.
- Logika state management dipusatkan menggunakan Alpine.js agar kode lebih terstruktur.
